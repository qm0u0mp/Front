### React 설치
- node.js 설치
- node -v , npm -v 버전 확인으로 설치되었는지 확인

### React 실행
- npx create-react-app 프로젝트명 --template typescript
  
- 설치하려는 위치 확인하고
- npm install
- npm run start

### Build 실행
- npm run build:development
=> 코드 수정 시 다시 실행
=> powerShell에서 git pull

### 배포
- npm install -g serve (-g : 글로벌 영역)
- serve -s build (local 주소로 실행)
